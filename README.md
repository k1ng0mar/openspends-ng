# OpenSpends NG

Track Nigerian government budget allocation & spending — by arm, ministry, and location.

## Tech Stack
- **Backend**: FastAPI + PostgreSQL/PostGIS
- **Frontend**: React + TypeScript + Vite + Tailwind + Mapbox

## Project Structure
```
OpenSpends-NG/
├── backend/    # FastAPI app
├── frontend/   # React app
└── docs/       # Data sources, plans, schema docs
```
